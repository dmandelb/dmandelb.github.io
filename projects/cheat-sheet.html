SQL Access Cheatsheet

How to access data in a table: Basics

  • Semicolons must be added to the end of any completed statement. If there is no semicolon, SQL will think that you mean to continue on the next line.
  • Specify the data type (column) you want to access using the SELECT command 
    SELECT full_name
  • Specify the table from which you want your data using the FROM command 
    SELECT full_name
    FROM hp_verse_characters;
  • Use the WHERE clause to filter results 
    SELECT full_name
    FROM hp_verse_characters
    WHERE house="Hufflepuff";
  • Like elsewhere, AND and OR let you add additional conditions to your modifiers
    SELECT full_name
    FROM hp_verse_characters
    WHERE house="Hufflepuff"
    OR house="Ravenclaw";
  • You may use the * character to indicate that you want all columns from a given table
    SELECT *
    FROM hp_verse_characters;
  • Specify that you want no duplicate values returned by including the DISTINCT modifier 
    SELECT DISTINCT house
  • Dictate the order in which your selected data is displayed with the ORDER BY command, and specifying ascending or descending with ASC or DESC 
    SELECT full_name
    FROM hp_verse_characters
    ORDER BY id ASC;
  • The LIKE operator allows you to feed in a pattern with a wildcard. If, for example, you wanted to search for any student with a first name of Harry, you could use 
    SELECT full_name
    FROM hp_verse_characters
    WHERE full_name LIKE "Harry %";

Joining

  • An INNER JOIN or just JOIN will return only the rows of the specified columns that fit the criteria specified
  • A FULL OUTER JOIN will return all rows of the specified columns, no matter whether or not there is a match to the specified criteria in any given row
  • A LEFT JOIN will return all rows of the left column, but only the rows of the right column in which there is a match to the specified criteria
  • A RIGHT JOIN will return all rows of the right column, but only the rows of the left column in which there is a match to the specified criteria
